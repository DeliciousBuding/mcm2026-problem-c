# 功能说明（中文）

## 1. 数据建模功能
- **百分比规则采样**：使用 Dirichlet 生成候选 fan share，并用极简约束筛选可行样本。
- **不确定性量化**：输出后验均值与 95% 区间宽度（HDI 近似）。
- **错误淘汰概率**：统计被淘汰者非最低 fan share 的概率热图。
- **并行支持**：按赛季并行采样与后验估计，提升高采样规模下的速度。

## 2. 规则切换推断
- 以各季可行性证据构建简化 HMM，输出 rank+save 后验概率曲线。
- 使用 bootstrap 子样本估计置信带，输出规则切换概率区间图。

## 3. 机制评估
- 计算冲突指数（Kendall τ）、观众能动性、评委一致性与稳定性。
- 评估流程已向量化，支持更高采样规模下的快速统计。
- 输出 DAWS 与基线机制的雷达图及二维 Pareto 权衡图（agency vs stability，颜色=integrity）。
- 生成机制对比条形图，便于快速对比关键指标。
- 生成决赛阵容流向图（Alluvial-like）。

## 4. 影响因素分析
- Judges vs Fans 的双模型（默认快速版 OLS）。
- 输出职业舞伴效应森林图与特征效应散点。
- 输出争议人物周序密度图（Ridgeline 近似）。

## 5. 预测辅助
- 使用 GBDT 做淘汰预测并给出分季 AUC 曲线。

## 6. 论文集成
- 自动生成 `paper/summary_metrics.tex`，在 LaTeX 内引用数值。
- 生成的图表直接嵌入 `paper/main.tex`。
- 中文报告版本为 `paper/main_zh.tex`，与英文内容同步。

## 7. 可信度与规模实验
- Fast vs Strict 校验：对比近似采样与严格约束结果，输出误差与一致性指标。
- 规模实验：不同 N_PROPOSALS 下记录时间、误差、稳定性与冲突指数，生成拐点图。
