## B. 算法结构优化（高优先级）
1. 增加“HDI 分布图”

   - 展示不确定性集中在哪些周；
   - 输出：`fig_hdi_distribution.pdf`

2. Fast vs Strict 校验（精度可信度）
   - 以严格约束(`percent_constraints_ok`)作为近似精度基准；
   - 采样同规模下对比 fan_share 均值/HDI；
   - 输出：`fig_fast_vs_strict.pdf` + MAE 统计行；
   - 验收：给出“误差 < 某阈值”或“误差随规模下降”的结论。
   - 额外验收（必须）：
     - Top-k 一致率：各周“粉丝最低/后两名”一致率；
     - 机制结论保持性：fairness/agency/deficit 的差异 < 阈值或趋势一致。

3. 规则切换不确定性区间
   - 给每季 rank 概率加置信带（bootstrap 或子样本法）；
