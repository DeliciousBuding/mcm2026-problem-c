## 主要问题（高优先级）
1. **坏图/空图/信息缺失图**：三角图几乎空白、alluvial 含义不明、多图缺 legend 与编码说明。
2. **图文不一致**：规则切换图的曲线与 caption 结论不一致。
3. **指标逻辑风险**：Kendall τ 作为机制公平性轴不敏感，trade-off 论证站不稳；稳定性定义需同机制噪声对比。
4. **DAWS 改进论证偏弱**：文案提升过满，但图与数值支撑不足。
5. **数据清洗痕迹明显**：pro dancer 名称出现“脏标签”。
6. **排版瑕疵**：References 重复标题。
